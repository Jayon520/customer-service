## 第一步：上传到GitHub（2分钟）

### 方式A：网页上传（最简单）

1. 访问 https://github.com/new
2. 创建仓库：
   - 仓库名：`customer-service`
   - 选择 **Private**（私有）
   - 点击 "Create repository"

3. 点击 "uploading an existing file"

4. 拖拽上传以下所有文件：
   ```
   ✅ main.py
   ✅ config.py
   ✅ database.py
   ✅ models.py
   ✅ init_data.py
   ✅ requirements.txt
   ✅ Procfile
   ✅ runtime.txt
   ✅ railway.json
   ✅ .env.example
   ✅ start.sh
   ✅ knowledge_sample.json
   ✅ README.md
   ✅ services/ 文件夹（包含4个.py文件）
   ```

5. 点击 "Commit changes"

## 第二步：在Railway部署（3分钟）

1. **登录Railway**
   - 访问 https://railway.app/
   - 点击 "Start a New Project"
   - 选择 "Login with GitHub"

2. **部署代码**
   - 选择 "Deploy from GitHub repo"
   - 选择 `customer-service` 仓库
   - 点击 "Deploy Now"
   - 等待2-3分钟构建

3. **添加数据库**
   - 点击右上角 "+ New"
   - 选择 "Database" → "Add PostgreSQL"

4. **配置环境变量**
   
   点击你的服务 → "Variables" → 添加以下变量：
   
   ```
   WECHAT_CORP_ID = 你的企业ID
   WECHAT_AGENT_ID = 你的应用AgentId
   WECHAT_SECRET = 你的应用Secret
   WECHAT_TOKEN = 你的回调Token
   WECHAT_ENCODING_AES_KEY = 你的回调密钥
   DEEPSEEK_API_KEY = 你的DeepSeek密钥
   DATABASE_URL = ${{Postgres.DATABASE_URL}}
   ```

5. **生成域名**
   - Settings → "Generate Domain"
   - 获得地址：`https://xxxxx.up.railway.app`

---

## 第三步：配置企业微信（1分钟）

1. 登录企业微信后台：https://work.weixin.qq.com/

2. 进入你的应用 → 设置API接收

3. 填写回调URL：
   ```
   https://你的Railway域名/wechat/callback
   ```

4. 点击保存 → 验证成功！
